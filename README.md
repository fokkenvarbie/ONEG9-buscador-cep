<h1>Sistema de Busca de CEP</h1>

Um programa em Java que permite consultar informações de endereço a partir de um CEP informado pelo usuário. A aplicação realiza a validação da entrada, faz a consulta dos dados e exibe as informações de forma organizada no console. O sistema utiliza princípios de orientação a objetos para separar responsabilidades e manter uma estrutura clara e reutilizável.

- **Funcionalidades:**
  - **Consulta de CEP:** O usuário informa um CEP via console, que é utilizado para buscar os dados correspondentes de endereço.
  - **Validação de entrada:** O sistema verifica se o CEP informado está em um formato válido antes de realizar a consulta.
  - **Exibição de dados de endereço:** As informações retornadas, como logradouro, bairro, cidade e estado, são exibidas de forma legível para o usuário.
  - **Tratamento de erros:** O programa lida com exceções e situações inválidas, exibindo mensagens claras em caso de falha na consulta.
  - **Interação via console:** Toda a comunicação com o usuário ocorre pelo terminal, permitindo acompanhar o fluxo da aplicação em tempo real.
  - **Uso de orientação a objetos:** As informações de endereço são representadas por uma classe específica, enquanto a lógica de consulta é isolada em outra classe, garantindo organização e facilidade de manutenção.
